## Featured Project

### [GCP Production-Grade Infrastructure](https://github.com/tomasnavb/gcp-wordpress-infra)

Production-grade cloud infrastructure deployed on GCP using Terraform.

**Architecture highlights:**
- Two VPCs with no public IPs — all traffic routed privately
- Cloud SQL with private connectivity
- HTTP Load Balancer + Cloud NAT for egress
- Identity-Aware Proxy (IAP) for secure access
- Automated backups via Cloud Functions with a disaster recovery bucket
- CI/CD pipeline with Cloud Build + GitHub for automated infrastructure deployment
- Terraform module structure following Google's recommended layout — networking, compute, database, IAM, storage
